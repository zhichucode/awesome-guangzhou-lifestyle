# 贡献指南

感谢你对 Awesome Guangzhou Lifestyle 的关注！欢迎通过以下方式贡献内容。

## 如何贡献

### 方式一：提交 Pull Request

1. Fork 本仓库
2. 创建你的分支 (`git checkout -b add-new-restaurant`)
3. 添加或修改内容
4. 提交你的更改 (`git commit -m '添加xxx餐厅'`)
5. 推送分支 (`git push origin add-new-restaurant`)
6. 创建 Pull Request

### 方式二：提交 Issue

如果你不想直接修改，可以通过 Issue 提出建议：
- 推荐新的餐厅/店铺
- 指出过时信息
- 提出分类建议

## 内容规范

### 添加新条目

请确保包含以下信息：
- **店名**：必填
- **地址/位置**：必填，可标注地铁站
- **特色/推荐菜**：简明扼要
- **人均消费**：可选
- **营业时间**：可选

### 示例格式

```markdown
- **店名** - 简短描述，特色推荐
```

### 推荐标准

- 品质可靠、口碑良好
- 具有广州特色或代表性
- 性价比高或体验独特

### 不推荐内容

- 广告推广性质过强
- 信息不完整
- 已关闭或搬离的店铺

## 分类指南

| 类别 | 包含内容 |
|------|----------|
| 美食餐厅 | 餐厅、cafe、小吃、夜宵 |
| 休闲娱乐 | 咖啡店、酒吧、密室、健身房 |
| 文化艺术 | 公园、博物馆、剧院、古迹 |
| 交通出行 | 地铁、公交、单车、打车 |
| 学习交流 | 讲座、读书会、语言交流 |

## 审核标准

提交的内容会从以下方面审核：
- 信息准确性
- 与现有内容的互补性
- 分类是否合适
- 描述是否客观真实

## 联系方式

如有疑问，欢迎提交 Issue 或 PR。
