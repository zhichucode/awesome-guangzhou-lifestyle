## 美食餐厅

### 早茶点心

广州人"饮早茶"的传统延续至今，以下是一些值得一试的茶楼：

- **陶陶居** - 百年老字号，环境典雅，点心精致
- **点都德** - 连锁品牌，品质稳定，虾饺皇必点
- **广州酒家** - 老牌名店，早茶下午茶都不错
- **莲香楼** - 传统茶楼，体验老广州味道
- **银记肠粉店** - 肠粉配艇仔粥，早餐首选

### 粤菜餐厅

正宗粤菜，讲究清而不淡、鲜而不俗：

- **惠食佳** - 米其林推荐，啫啫煲一绝
- **炳胜品味** - 粤菜标杆，脆皮叉烧必点
- **莲江酒家** - 藏在村里的米其林
- **空中一号** - 高端粤菜，适合商务宴请
- **大同酒家** - 老字号，怀旧粤菜

### 街头小吃

广州街头隐藏着无数美食宝藏：

- **陈添记** - 鱼皮爽脆，人均30元
- **伍湛记** - 及第粥、艇仔粥
- **南信牛奶甜品** - 双皮奶、姜撞奶
- **顺记冰室** - 传统雪糕，榴莲雪糕
- **佳记面店** - 竹升面、云吞面
- **生记肠粉** - 布拉肠，软糯可口
- **阿婆牛杂** - 街边牛杂，烟火气足

### 夜宵大排档

广州夜宵文化浓厚，大排档是深夜食堂：

- **潮汕毒药** - 生腌海鲜，宵夜网红店
- **榕树头鸡煲** - 顺德火焰醉鸡煲
- **贞姨大排档** - 炒蟹、炒螺
- **食为先夜宵** - 炒牛河、炒田螺
- **标记大排档** - 烧生蚝、烤串
- **芳村阿婆牛杂** - 夜宵档牛杂

### 火锅烤肉

火锅烤肉也是广州人的日常：

- **捞王锅物料理** - 猪肚鸡火锅
- **海底捞火锅** - 服务标杆
- **潮汕牛肉火锅** - 汕头八合里、海银海记
- **椰子鸡火锅** - 凤园椰珍、椰子不语
- **烤肉** - 烤肉连锁店、广州本地烤肉店

### 日料西餐

广州日料西餐选择丰富：

- **美登利寿司** - 活造寿司
- **鮨· SUSHI** - 板前寿司
- **元色寿司** - 性价比高
- **Amuse** - 法餐日作
- **TIAGO** - 意大利菜
- **极旨烤肉** - 烧肉放题

## 交通出行

### 地铁公交

广州公共交通发达，地铁是最便捷的出行方式：

- **地铁**：已开通15条线路，票价按里程计算
- **羊城通**：交通卡，地铁公交通用，可刷公交
- **广州地铁APP**：实时线路图、出行规划
- **公交**：覆盖全市，票价2元（BRT段统一2元）
- **有轨电车**：海珠有轨电车，观光好选择

> 提示：下载"广州地铁"APP可查询实时到站信息，高峰期建议避开3号线、5号线。

### 共享单车

广州共享单车品牌众多：

- **美团单车** - 黄色，量大
- **哈啰单车** - 蓝色，舒适
- **青桔单车** - 绿色，调度多
- **广州公共自行车** - 政府运营

> 提示：地铁站口单车较多，建议停放在规定区域避免被收。

### 打车租车

- **滴滴出行** - 网约车主流平台
- **曹操出行** - 新能源车为主
- **出租车** - 扬招或电召
- **神州租车** - 长租自驾
- **一嗨租车** - 短租自驾
